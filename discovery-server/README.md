# Eureka Discovery Server

Eureka Service Registry running on port `8761`. It provides service discovery, instance heartbeat tracking, and load balancing coordinates.

## Dashboard
Open [http://localhost:8761](http://localhost:8761) to view the Eureka status console. This page lists all registered service names (e.g. `AUTH-SERVICE`, `NOTIFICATION-SERVICE`, `API-GATEWAY`) along with their dynamic IP addresses.
